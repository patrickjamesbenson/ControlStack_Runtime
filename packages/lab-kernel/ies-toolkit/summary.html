Normalise (ordered recipe)

2Level tilt-correct so the peak sits at nadir - skip if an off-nadir peak is intended accept dimensions first
3Symmetrise enforce azimuth symmetry & record the imbalance - skip if the asymmetry is intended
4Interpolate resample to the standard grid (default 181×25, adjustable)
5Pad hemisphere zero the non-emitting hemisphere (after interpolate)
6Reduce to 1 mm scale candela + power + length to the 1 mm base

Orient (each is undoable)

Roll - about vertical(plan view) °
Tilt about WIDTH axis(side view: beam down→up) °
Tilt about LENGTH axis(end-on view: beam swings sideways) °
Flip 180°(down↔up)

Match the button to the view: Tilt about the width axis see-saws the length - the beam rotates down→up in the side view (this is your "end-on, rotate about the horizontal line, 180° = up"). Tilt about the length axis spins it end-on - the beam swings sideways in the end-on view. Both reach straight up at 180°.
